## Running the project

1. [Fork](https://github.com/avie-dev/spaceslounge/fork) the project

2. Clone the repo:

   ```console
   $ git clone https://github.com/<your-github-username>/spaceslounge.git
   ```

3. Navigate to the cloned directory:

   ```console
   $ cd spaceslounge
   ```

4. Install dependencies:

   ```console
   $ yarn
   ```

5. Run the project

   ```console
   $ yarn dev
   ```
